📁 Dove mettere admin.html nel repository
La struttura consigliata del tuo repository nicolamartucci.github.io è:
nicolamartucci.github.io/ ├── tuttoapricena/ │ ├── index.html ← sito pubblico │ ├── admin.html ← questo CMS (pannello admin) │ └── assets/ │ └── data.json ← dati pubblicati dal CMS └── (altri file del repo)
✅ admin.html va nella cartella tuttoapricena/ — si accede da https://nicolamartucci.github.io/tuttoapricena/admin.html
✅ data.json viene creato automaticamente dal CMS al primo "Pubblica"
✅ La cartella assets/ deve esistere nel repo (anche vuota, aggiungi un file .gitkeep dentro)

📖 Guida passo-passo
1
Crea la struttura cartelle nel repo
Su github.com → il tuo repo → Add file → Create new file → scrivi tuttoapricena/assets/.gitkeep → Commit
2
Carica admin.html nel repo
Su github.com → tuttoapricena/ → Add file → Upload files → carica admin.html → Commit
3
Genera il token GitHub
Settings → Developer settings → Fine-grained tokens → Genera con permesso Contents: Read and Write sul repo specifico
4
Configura e testa qui
Incolla il token sopra → Salva configurazione → Testa connessione (deve dire ✓ OK)
5
Pubblica!
Vai su Dashboard o qualsiasi pagina → clicca "Pubblica sul sito" in alto a destra → il sito si aggiorna in 10–30 sec
6
Aggiungi lo snippet al sito
