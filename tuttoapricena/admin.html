GitHub & Sincronizzazione

Collega il CMS al repo per pubblicare live

✓ Impostazioni salvate
✕
Configurazione repository
Genera su github.com → Settings → Developer settings → Personal access tokens → Fine-grained. Permessi: Contents: read & write
Come funziona
  1. Inserisci il token GitHub con permessi Contents: read & write sul tuo repo
  2. Il CMS legge e scrive il file tuttoapricena/assets/data.json direttamente nel repo
  3. GitHub Pages aggiorna il sito in pochi secondi dopo ogni "Pubblica"
  4. Aggiungi nel tuo index.html lo script di lettura dati (vedi sotto)
