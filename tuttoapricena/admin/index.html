GitHub Sync
Pubblica i contenuti modificati sul sito con un clic
Configurazione repository
Il "Scarica data.js" funziona sempre, anche senza token
Istruzioni passo per passo
1
Crea un Personal Access Token

GitHub.com → Settings → Developer settings → Personal access tokens → Tokens (classic) → Generate new token → seleziona scope repo → copia il token

2
Inserisci token e repository

Incolla il token nel campo sopra e scrivi il tuo repository nel formato username/tuttoapricena

3
Testa la connessione

Clicca "Test connessione" per verificare che tutto funzioni prima di pubblicare

4
Modifica i contenuti e pubblica

Fai le tue modifiche nel CMS, poi clicca "Pubblica sul sito". Il file data.js viene aggiornato su GitHub e il sito si rinnova in 1-2 minuti

Se "Pubblica" dà errore di rete: usa "Scarica data.js", poi caricalo manualmente su GitHub nel percorso tuttoapricena/assets/js/data.js
