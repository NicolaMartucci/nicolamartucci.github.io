Il token viene salvato automaticamente nel browser (crittografato). Non dovrai reinserirlo.
Come si usa
1

Crea token — GitHub.com → Settings → Developer settings → Tokens (classic) → Generate → scope repo

2

Configura — incolla token e repository. Vengono salvati automaticamente nel browser.

3

Pubblica — clicca "Pubblica sul sito". Il sito si aggiorna in 1-2 minuti.

Se la rete dà errore, usa "Scarica data.js" e caricalo manualmente su GitHub → tuttoapricena/assets/js/data.js
