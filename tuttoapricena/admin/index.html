Configurazione repository
Istruzioni
  1. Su GitHub vai in Settings → Developer settings → Personal access tokens → Classic
  2. Crea un token con scope repo (accesso completo ai repository)
  3. Incollalo nel campo sopra insieme al nome del tuo repository
  4. Modifica contenuti nel CMS (notizie, eventi, locali...)
  5. Clicca Pubblica sul sito: il file data.js viene aggiornato su GitHub
  6. Con GitHub Pages attivo, il sito si aggiorna automaticamente entro 1-2 minuti
