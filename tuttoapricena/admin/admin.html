⬇️ Esporta Dati Sito

Scarica un file tuttoapricena-data.json con tutti i contenuti del sito (notizie, eventi, sponsor, impostazioni, layout). Carica questo file su GitHub nella cartella assets/ per aggiornare il sito.

📋 Istruzioni GitHub
  1. Esporta il file tuttoapricena-data.json
  2. Vai su GitHub nel repo del sito
  3. Naviga nella cartella assets/
  4. Carica il file (drag & drop o "Add file")
  5. Il sito leggerà automaticamente i nuovi dati
  6. Per il logo o immagini: carica prima le immagini in assets/img/ e usa percorsi relativi nel CMS
